
## 🚀 本地运行指南（CMD命令行）

1️⃣ **创建虚拟环境并安装依赖**
```
python -m venv venv
```
* Windows
```
venv\Scripts\activate
```
* Linux/MacOS
```
source venv/bin/activate
```
```
pip install -r requirements.txt
```

2️⃣ **启动预测Web应用**

如果你不激活环境，而是直接调用虚拟环境中的命令：

/Users/自己的用户名/venv/bin/streamlit run app.py

```
streamlit run app.py
```
